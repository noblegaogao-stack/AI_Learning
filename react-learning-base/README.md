# React TypeScript 学习项目

这是一个专为刚入门的前端开发工程师设计的React + TypeScript学习项目，包含了13个核心知识点的demo，每个demo都有详细的注释和讲解，帮助你快速掌握React的所有用法，成为高级开发者。

## 项目结构

```
react-learning-base/
├── src/
│   ├── App.tsx          # 主应用组件，包含所有demo
│   ├── LazyComponent.tsx # 懒加载组件
│   ├── index.css        # 全局样式
│   └── main.tsx         # 应用入口
├── index.html           # HTML入口
├── package.json         # 项目配置
├── tsconfig.json        # TypeScript配置
├── tsconfig.node.json   # TypeScript Node配置
└── vite.config.ts       # Vite配置
```

## 项目包含的知识点

1. **基础组件和Props类型定义** - 学习如何定义组件的Props接口，包括必选和可选属性
2. **useState Hook** - 学习如何使用useState管理组件状态
3. **useEffect Hook** - 学习如何处理副作用，如数据获取
4. **useContext Hook** - 学习如何使用Context API进行状态管理
5. **自定义Hook** - 学习如何创建和使用自定义Hook
6. **类型断言和类型守卫** - 学习如何处理联合类型
7. **泛型组件** - 学习如何创建可重用的泛型组件
8. **条件渲染** - 学习如何根据条件渲染不同的内容
9. **列表渲染** - 学习如何渲染列表数据
10. **事件处理** - 学习如何处理DOM事件
11. **表单处理** - 学习如何处理表单输入和提交
12. **错误边界** - 学习如何捕获和处理组件错误
13. **懒加载和代码分割** - 学习如何优化应用性能

## 如何使用这个项目

1. **安装依赖**：在项目目录中运行 `npm install`
2. **启动开发服务器**：运行 `npm run dev`
3. **构建项目**：运行 `npm run build`
4. **查看代码**：打开 `src/App.tsx` 文件，查看每个demo的实现和注释
5. **实验代码**：修改代码，观察变化，加深理解

## 学习建议

1. **逐个知识点学习**：从第一个demo开始，理解每个知识点的概念和实现
2. **运行和测试**：启动开发服务器，交互测试每个demo的功能
3. **修改和扩展**：尝试修改代码，添加新功能，巩固学习
4. **查阅文档**：结合React和TypeScript官方文档，深入理解每个知识点
5. **实践项目**：基于所学知识，尝试构建自己的项目

这个项目涵盖了React + TypeScript的核心知识点，通过学习和实践，你将能够掌握React的基本用法，并逐步成为高级开发者。
